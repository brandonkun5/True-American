One, Two, Three, JFK! FDR!

The cultured history of True American.

True American was a fictional game

It was originally played on New Girl, introduced in season 1, episode 20: Normal.

The entire point of the game is to confuse anyone watching or playing, while getting really drunk. Naturally, we found some rules online and were pumped to try it.

But it SUCKED

We never considered that a game designed to be confusing wouldn't be that much fun.

You basically walked in circles for an hour playing the same three mini games the whole time.

So we made our own version!

The new and improved game

Is every bit as ridiculous as the original game, but we added a lot of elements so it doesn't get boring if you play for a while.

Since our version requires special game cards, we made this site. That way, anyone that wants to play can just click "play" and enjoy True American with their friends.

The inaugural game

After we made some cards, we invited a few of our close friends over for the beta test on October 24, 2020.

They loved it, and the rest is history
